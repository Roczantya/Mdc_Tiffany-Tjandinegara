## Cara Instalasi

Berikut cara menjalankan proyek Flutter ini di komputer Anda:

1. **Clone repository:**
   ```bash
   git clone https://github.com/Roczantya/Mdc_Tiffany-Tjandinegara.git

2. **Pindah ke direktori proyek**
     ```bash
     cd nama-proyek
3. **Instal depedencies**
     ```bash
     flutter pubget
4. **jalankan aplikasi**
     ```bash
     flutter run
